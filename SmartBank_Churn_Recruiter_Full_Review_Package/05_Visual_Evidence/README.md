# SmartBank Customer Churn Prediction - Recruiter Full Review Package

This folder is organized so a recruiter, hiring manager, or technical reviewer can check the full project without searching through the original workspace.

## Open First

Start with:

`00_OPEN_FIRST/Recruiter_Review_Guide.md`

That file explains the project, the strongest evidence to review, and what each folder contains.

## Folder Map

- `00_OPEN_FIRST` - recruiter guide, review checklist, and file index.
- `01_Final_Reports` - polished Word and Markdown reports.
- `02_Data` - original dataset files and data notes.
- `03_Code_and_Reproducibility` - modeling script, requirements, notes, and run instructions.
- `04_Results_Tables` - CSV outputs for metrics, thresholds, feature importance, SHAP importance, and risk scores.
- `05_Visual_Evidence` - charts used to support the analysis.
- `06_Model_Artifacts` - saved model files and metadata.
- `07_Project_Context` - optional workflow context from the project.

## What Was Not Included

Installed dependency folders such as `.python_packages` are intentionally excluded because they are large machine-local files. The required libraries are listed in `03_Code_and_Reproducibility/requirements.txt`.
