# Project Review Checklist

Use this checklist to confirm the project can be reviewed end to end.

## Reports

- [ ] Final business/report document is included.
- [ ] Modeling report is included in Word format.
- [ ] Modeling report is included in Markdown format.

## Data

- [ ] Main CSV dataset is included.
- [ ] Excel dataset is included.
- [ ] Data notes explain target column and important fields.

## Code

- [ ] Main modeling script is included.
- [ ] Requirements file is included.
- [ ] Run instructions are included.
- [ ] Modeling notes are included.

## Results

- [ ] Model performance comparison is included.
- [ ] Cross-validation summary is included.
- [ ] Imbalance strategy comparison is included.
- [ ] Hyperparameter tuning results are included.
- [ ] Operating threshold analysis is included.
- [ ] Feature importance table is included.
- [ ] SHAP importance table is included.
- [ ] Test-set churn risk scores are included.
- [ ] Top 50 high-risk customers are included.

## Visual Evidence

- [ ] Churn distribution chart is included.
- [ ] Cross-validation comparison chart is included.
- [ ] Confusion matrix chart is included.
- [ ] ROC curve chart is included.
- [ ] Random Forest feature importance chart is included.
- [ ] SHAP summary chart is included.

## Model Artifacts

- [ ] Saved Random Forest model artifact is included.
- [ ] Modeling metadata is included.

## Context

- [ ] Optional workflow context is included.
- [ ] Excluded machine-local dependency folders are documented.
